# Расписания бэкапов

В этом каталоге хранятся файлы расписаний для бэкапов сервисов.

## Формат файлов

Файлы расписаний должны быть в формате cron и иметь расширение `.cron`.

Пример файла `daily.cron`:
```
0 2 * * *  # Ежедневно в 02:00
```

## Стандартные расписания

- `hourly.cron` - каждый час
- `daily.cron` - ежедневно
- `weekly.cron` - еженедельно
- `monthly.cron` - ежемесячно

## Использование

Backup Manager автоматически считывает расписания из этого каталога и применяет их к сервисам в соответствии с настройками в `service.yml`.
